# nowDownTime+
Сайт по продаже настольно-ролевых игр.

Как установить сайт в 1С Битрикс: Малый-бизнес:
1. Скачать папку local
2. Перенести её в рабочую область, а именно: C:\Open Server Panel\home(или domains)\(Название сайта)
3. Дальше перейти в админ-панель сайта -> Настройки -> Настройки продукта -> Шаблоны сайтов.
   У вас сразу должен отобразиться nowDownTime в списках. Если всё хорошо пропустите следующий пункт.
4. Если вы не видите в шаблонах сайта этот шаблон то сдлаейте следующее: Откройте папку local\templates и скопируйте файл MySite, после чего отправляйтесь в папку C:\Open Server Panel\home(или domains)\(Название сайта)\bitrix\templates и удалите остальные Шаблоны и вставляйте папку MySite. Затем возвращайтесь в Шаблоны сайтов (Пункт 3).
5. Дальшей перейдите в админ-панель сайта -> Настройки -> Настройки продукта -> Список сайтов. И начните редактировать тот сайт что назначен как "По умолчанию: Да", прокрутив страницу вниз вы увидит Шаблон сайта, в первом столбце Шаблон выберите nowDownTime+ и возвращайтесь на сайт. Всё готово.
