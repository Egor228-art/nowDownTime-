# nowDownTime+
Интернет-магазин настольных игр

## Требования
- PHP 7.4+
- MySQL 5.7+
- Bitrix CMS

## Установка
1. Скопируйте все файлы проекта в корневую директорию сайта
2. Создайте базу данных MySQL
3. Скопируйте `settings.example.php` в `settings.php` и укажите свои данные для подключения к БД
4. Запустите установку Bitrix через веб-интерфейс

## Структура проекта

- `/auth` - авторизация
- `/catalog` - каталог товаров
- `/personal` - личный кабинет
- `/local` - локальные компоненты и шаблоны
- `ajax.php` - AJAX обработчики для модальных окон

## Особенности
- Модальные окна входа и регистрации
- AJAX авторизация без перезагрузки
- Валидация полей в реальном времени
- Адаптивный дизайн

## Безопасность
Все конфиденциальные данные исключены из репозитория:
- Пароли к БД
- Ключи доступа
- Файлы конфигурации
- Папка upload с пользовательским контентом
- Кэш и логи
